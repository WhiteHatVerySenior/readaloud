# ReadAloud 📖

An interactive read-aloud platform for Class 7 students with speech recognition, proficiency tracking, and a teacher dashboard.

## Live App
👉 **[Open ReadAloud](https://YOUR-USERNAME.github.io/readaloud)**

> Replace `YOUR-USERNAME` with your GitHub username after deploying.

## Features
- 🎤 Real-time speech recognition — words highlight as you read
- 📊 Live accuracy % and words-per-minute tracking
- 🏆 Automatic level detection (Beginner → Developing → Fluent → Advanced)
- 👩‍🏫 Teacher dashboard with class overview, student roster, and story assignment
- 📚 4 stories across English, Science, and EVS
- 🌙 Works on any device with Chrome or Edge

## How to deploy (5 steps)
1. Create a free account at [github.com](https://github.com)
2. Click **New repository** → name it `readaloud` → make it **Public** → click **Create**
3. Drag `index.html` and `README.md` into the repository file area and click **Commit**
4. Go to **Settings → Pages → Source → Deploy from branch → main → / (root) → Save**
5. Wait 1 minute → your app is live at `https://YOUR-USERNAME.github.io/readaloud`

Pure HTML + CSS + JavaScript. No frameworks, no build step, no server needed.
Uses the browser's built-in Web Speech API for speech recognition.
